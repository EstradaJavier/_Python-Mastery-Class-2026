# Symbols & Notation Reference Guide

Comprehensive reference for symbols used in Python, programming, terminal, and regex.

## 📌 Mathematical & Logical Operators

| Symbol | Name                        | Common Use Case |
|--------|-----------------------------|-----------------|
| `+`    | Plus                        | Addition, concatenation |
| `-`    | Minus                       | Subtraction |
| `*`    | Asterisk                    | Multiplication, unpacking (`*args`) |
| `/`    | Forward Slash               | Division (float) |
| `//`   | Floor Division              | Integer division |
| `%`    | Modulo                      | Remainder, string formatting |
| `**`   | Exponent                    | Power (`2 ** 3`) |
| `=`    | Assignment                  | Assign value |
| `==`   | Equality                    | Comparison |
| `!=`   | Not Equal                   | Comparison |
| `>` / `<` | Greater / Less Than     | Comparison |
| `>=` / `<=` | Greater/Less or Equal | Comparison |

## 🧩 Data Structures & Python Specific

| Symbol | Name                        | Use Case |
|--------|-----------------------------|----------|
| `[]`   | Square Brackets             | Lists, indexing, slicing |
| `{}`   | Curly Braces                | Dictionaries, sets |
| `()`   | Parentheses                 | Tuples, function calls |
| `:`    | Colon                       | Slicing, type hints, dicts |
| `->`   | Arrow                       | Function return type hint |
| `:=`   | Walrus Operator             | Assignment expression (Python 3.8+) |
| `*args` | Star args                   | Variable positional arguments |
| `**kwargs` | Double star kwargs       | Variable keyword arguments |
| `@`    | At Symbol                   | Decorators |
| `__`   | Dunder (Double Underscore)  | Magic methods (`__init__`, `__str__`) |

## 🔍 Regular Expressions (Regex) Symbols

| Symbol | Name                        | Use Case |
|--------|-----------------------------|----------|
| `.`    | Dot                         | Any character except newline |
| `^`    | Caret                       | Start of string |
| `$`    | Dollar Sign                 | End of string |
| `*`    | Asterisk                    | Zero or more occurrences |
| `+`    | Plus                        | One or more occurrences |
| `?`    | Question Mark               | Zero or one occurrence |
| `\|`   | Pipe                        | Alternation (OR) |
| `[]`   | Character Class             | Match any character inside |
| `()`   | Capturing Group             | Group and capture |
| `\d`   | Digit                       | Any digit (0-9) |
| `\w`   | Word Character              | Letter, digit, or underscore |
| `\s`   | Whitespace                  | Space, tab, newline |

## 💻 Terminal / Shell Symbols

| Symbol | Name                        | Use Case |
|--------|-----------------------------|----------|
| `cd`   | Change Directory            | Navigate folders |
| `ls`   | List                        | Show files |
| `pwd`  | Print Working Directory     | Show current path |
| `&&`   | AND operator                | Run command if previous succeeds |
| `\|`   | Pipe                        | Pass output to another command |
| `>`    | Redirect                    | Save output to file |
| `>>`   | Append Redirect             | Append output to file |
| `2>`   | Stderr Redirect             | Redirect error output |
| `;`    | Command Separator           | Run multiple commands |
| `~`    | Tilde                       | Home directory |
| `..`   | Double Dot                  | Parent directory |
