This page is designed for beginner coders. It starts simple, then builds up through variables, strings, math, lists, loops, conditions, dictionaries, functions, tables, and small interactive examples.

How to use this page

Read one section at a time. Then open the Python code at the bottom, change a value, save the file, refresh the browser, and see what changes.

Best practice: make one small edit at a time so you can clearly see what each change does.
